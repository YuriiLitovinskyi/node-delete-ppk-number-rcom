Утиліта для видалення ппк з РКОМу за номером.
Запускається на хості з РКОМом.
РКОМ та Монго повинні бути запущені.

Може використовуватись, якщо потрібно видалити конкретний номер приладу з РКОМу, або виправити помилку з дублями ппк.

У другому випадку - після видалення відправити з ПЦС команду Опитування на номер видаленого ппк (або рестартувати ппк по живленню).
Це необхідно для того, щоб актуалізувати поточний стан приладу для подальшої нормальної роботи з мобільними додатками.
